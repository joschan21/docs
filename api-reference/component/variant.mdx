---
title: 'Variant'
---

The `<Variant />` component is used to wrap the content of each variant in your experiement. The content of a variant can be either a server or a client component.

## Props

<ResponseField name="name" type="string" required>
  The name for your variant. This will be shown in your Splitter dashboard.
</ResponseField>

<ResponseField name="children" type="JSX Elements">
  Any valid JSX elements.
</ResponseField>
